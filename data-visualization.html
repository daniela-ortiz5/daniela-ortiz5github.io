Data Visualization

A collection of maps created with Datawrapper and QGIS for reporting projects. These projects use statistics and structured information to make reporting easier to understand.

Chart Datawrapper

A chloropleth map showing canopy coverage loss in Alachua County, Florida, which was built using a combination of Datawrapper and QGIS. This map was created for an upcoming WUFT story about tree loss in the county.

Map Datawrapper

A locator map showing selected running routes in Gainesville, Florida, based on reporter-collected data. The lines were drawn using geojson.

Chart Datawrapper

Chloropleth map showing opt-in to the K-12 school guardian program by county in Florida. This map was used in my WUFT-published story about the school guardian program.
